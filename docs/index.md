# Welcome to PSPDevNotes

创建这个网站的初衷，一个是想研究古早设备，另一个是想作为学习的契机。通过一个近20年前的设备，学习计算机知识。

## Goals

* 熟悉Linux开发环境，Bash Shell的有关知识
* 熟悉GNU的一系列工具，了解大型C/C++项目如何组织
* 练习使用git和GitHub，尝试使用CI/CD工具
* 学习计算机系统基础，逐渐从其他语言到C语言和汇编过渡
* 从 GNU Make 到 CMake 
* 了解不同平台的差异
